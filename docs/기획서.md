# 3초 생존 게임 기획서

## 1. 게임 개요
### 컨셉
- **장르**: 캐주얼 액션, 미니게임 컬렉션
- **플랫폼**: 모바일 웹 (반응형)
- **플레이 타임**: 1회 3초, 세션당 1~5분
- **핵심 메커니즘**: 무작위 미션을 3초 안에 성공하면 생존, 실패하면 게임오버
- **테마**: CRT 그린톤 레트로 아케이드

### 게임 플로우
```
홈 화면 → START → 미션 제시 (0.5초) → 플레이어 액션 (2.5초) → 성공/실패 판정 → 다음 미션 or 게임오버 → 결과 화면
```

## 2. 핵심 시스템

### 2.1 미션 시스템
각 미션은 다음 요소로 구성:
- **미션 텍스트**: 간결한 지시문 (한/영 지원)
- **미션 타입**: 135종의 고유 미니게임
- **난이도**: EASY, NORMAL, HARD
- **시각적 요소**: 색상, 아이콘, 애니메이션

매 라운드 미션 1개가 제시되며, 3초 내에 성공해야 다음 라운드로 진행.

### 2.2 입력 타입
- **TAP**: 단일 터치
- **SWIPE**: 방향 스와이프 (UP, DOWN, LEFT, RIGHT)
- **LONG_PRESS**: 1초 이상 롱프레스
- **MULTI_TAP**: 빠른 연타
- **DUAL_TAP**: 두 손가락 동시 터치

### 2.3 미션 카테고리 (135종)
- **시스템/해킹**: REBOOT, DEFRAG, KERNEL_PANIC, FIREWALL, BIOS_ERROR, SEGFAULT, BLUE_SCREEN 등
- **의료/생존**: HEARTBEAT, TRANSFUSE, ANTIDOTE, TOURNIQUET, QUARANTINE, DEFIBRILLATE, TRIAGE 등
- **전투/전술**: GRENADE_PIN, BARRICADE, SCOPE, RELOAD, CLAYMORE_AIM, TRIPWIRE, BREACH_CHARGE 등
- **통신/신호**: MORSE, BEACON, BROADCAST, SEMAPHORE, SIGNAL_INTERCEPT, FREQUENCY_JAM 등
- **전자/회로**: SOLDER, CAPACITOR, VOLTAGE_MATCH, CIRCUIT_BRIDGE, FIRMWARE, TUBE_REPLACE 등
- **환경/탐색**: COMPASS, GEIGER, DUST_STORM, WATER_LEVEL, FORAGE, TUNNEL_DIG, THERMAL_SCAN 등

### 2.4 난이도 시스템
- **스코어 기반 난이도 증가**
  - 0~5점: EASY 미션 (기본 16종)
  - 6~15점: EASY + NORMAL 미션 (84종)
  - 16~30점: NORMAL 미션 확장
  - 31점+: 전체 135종 (HARD 포함)

### 2.5 피드백 시스템
- **성공 시**:
  - 효과음 (2톤 사인파 880Hz + 1320Hz)
  - 화면 흔들림 (300ms) + 파티클 이펙트 (12개)
  - +1 점수 표시

- **실패 시**:
  - 효과음 (2톤 스퀘어파 200Hz + 150Hz)
  - 화면 붉은 플래시
  - "FAIL" 텍스트 + 최종 점수

## 3. UI/UX

### 3.1 화면 구성
- **홈 화면**: 최고 점수 표시, START 버튼, 설정 메뉴
- **게임 화면**: 타이머 바, 점수, 미션 텍스트, 미션 컴포넌트, 페이즈 뱃지
- **결과 화면**: 최종 점수, 신기록 표시, 재시작/홈 버튼

### 3.2 설정
- BGM 볼륨 조절
- SFX 볼륨 조절
- 언어 전환 (한국어/영어)

### 3.3 점수 저장
- 최고 점수 (localStorage)
- 최근 50개 점수 히스토리

## 4. 기술 스택
- **프레임워크**: Vue 3 (Composition API)
- **빌드 도구**: Vite
- **언어**: TypeScript
- **라우팅**: Vue Router 4
- **스타일**: CSS3 (커스텀 변수, Flexbox, Grid, Animation)
- **오디오**: Web Audio API (SFX 합성 + BGM 스텝 시퀀서)
- **폰트**: Mulmaru (커스텀 WOFF2)
- **상태 관리**: Vue Reactivity (Ref, Computed)
- **i18n**: 커스텀 동적 패턴 기반 번역 시스템
